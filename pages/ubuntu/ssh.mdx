# SSH

>Installation

```shell
sudo apt update && sudo apt install openssh-server
```

>Check if is activated

```shell
sudo systemctl status ssh
```

>If not activated

```shell
sudo systemctl enable openssh-server
```

>In other location

```shell
ssh [user]@[ubuntu-server-ip]
```

>If SSH changed

```shell
ssh-keygen -R [ubuntu-server-ip]
```
